# 🎤 Voice Translator using Watsonx

## 📌 Description
This project is a voice-enabled AI translator that converts speech into text, translates it into another language using IBM Watsonx (FLAN-UL2 model), and converts the translated text back into speech.

It combines Speech-to-Text (STT), Large Language Models (LLM), and Text-to-Speech (TTS) to provide a complete voice translation experience through a web interface.

---

## 🚀 Features
- 🎤 Speech-to-Text conversion
- 🌍 Multilingual translation using LLM
- 🔊 Text-to-Speech output
- 💻 User-friendly web interface
- ⚡ Backend powered by Flask (Python)

---

## 🛠️ Technologies Used
- Python
- Flask
- IBM Watsonx AI (FLAN-UL2)
- Speech-to-Text API
- Text-to-Speech API
- HTML, CSS, JavaScript

---

---

## ⚙️ How it Works

1. User speaks or types input  
2. Speech is converted into text (STT)  
3. Text is sent to Watsonx LLM for translation  
4. Translated text is converted into speech (TTS)  
5. Output is shown and played to the user
